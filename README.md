## ⚙️ **一键安装、设置并启动 sing-box**

* 中文快速安装（使用默认配置）
* 部署多协议节点
* 支持Clash/V2rayN/Shadowrocket等所有科学上网客户端
* 智能判断操作系统来安装所需依赖和配置环境
* 自动配置防护墙、入站规则、过滤规则等
* 开启 BBR 加速
* 启动 sing-box 并设置开机自启
